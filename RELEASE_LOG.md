# Academia 发版记录

本文档记录 Academia 的版本发布历史和更新内容。

---

## 版本历史

### [1.0.0] - 2026-02-26

🎉 **首个正式版本发布**

#### ✨ 新功能

**资源管理**
- 支持 PDF 文档导入和预览
- 支持 Markdown 笔记创建和编辑
- 支持网页链接保存和管理
- 支持多种通用文件类型（图片、音频、视频、代码、文本等）

**标签系统**
- 层级标签结构，支持父子标签
- 自定义标签颜色
- 批量标签管理
- 标签计数显示

**预览功能**
- PDF 内嵌预览
- 图片预览（JPG/PNG/GIF/WebP）
- 音频播放器
- 视频播放器
- 文本/代码预览
- Markdown 实时渲染

**用户界面**
- macOS 风格原生标题栏
- 三栏布局（导航/列表/预览）
- 卡片式资源展示
- 响应式设计

#### 🧪 测试与 CI

- Jest 单元测试框架
- 资源类型完整测试覆盖
- GitHub Actions CI/CD 流程
- 三平台自动构建 (macOS/Windows/Linux)
- ESLint 代码检查
- 本地 CI 脚本支持

#### 🏗️ 架构

- 采用 Electron 28 框架
- 实现资源类型抽象（BaseResource 基类）
- 统一资源管理器（ResourceManager）
- 使用 electron-store 持久化存储

#### 📦 技术栈

- Electron 28.0.0
- electron-store 8.1.0
- uuid 9.0.0
- Jest 29.7.0
- ESLint 8.56.0

## 版本号规范

本项目遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/) 规范：

```
主版本号.次版本号.修订号

- 主版本号：不兼容的 API 变更
- 次版本号：向下兼容的功能新增
- 修订号：向下兼容的问题修复
```
